# multichat

[MultiTwitch](https://github.com/bhamrick/multitwitch), but for streamers. Allows smaller streamers without the ability to squad stream to easily keep up with not only their chat, but the chats of their co-streamers as well.

## Getting Started

First, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
